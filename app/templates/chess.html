Python Chess App

Chess is a fun game that I love to play. As a summer project I decided to try and build a chess app using python and decided to host it on a website using Flask. Currently the app only supports two player games. In the future I will try to create an algorithm so that you can play against the AI. If you would like to see the code, please visit my github: BushiDokaj
