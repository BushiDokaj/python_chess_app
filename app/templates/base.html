Python Chess App

{% include 'carousel.html' %}

Chess is one of the oldest and most complex games ever created. Chess is a game of strategy, tactics and calculation. As a child playing against my brother or my dad, it was a great exercise in problem solving. As a programmer it presents a great opportunity to develop my skill and experience. So, this summer, I built a chess app using Python and decided to host it on a website using Flask. Currently the app only supports two player games. In the future I will try to create an algorithm so that you can play against the computer. If you would like to see the code, please visit my github: BushiDokaj
